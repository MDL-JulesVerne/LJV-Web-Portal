I. Le site internet

ljv.fr a été créé d'après une initiative non-officielle d'un groupe de quelques lycéens du lycée Jules Verne. Ce site internet a été pensé comme un "portail", dans le but de faire le lien entre le site officiel du lycée et ses projets externes, gérés par des lycéens. Ces projets peuvent être hébergés sur la plateforme du site internet et sont liés à l’établissement ou à sa vie lycéenne.

L’organisation du domaine « ljv.fr » a été pensée afin d’offrir une certaine visibilité à tous les projets qui lui sont liés, ainsi qu’au site officiel du lycée.

II. Les projets

La plateforme a pour but de permettre l’hébergement de projets de lycéens sur Internet. Pour le moment, la plateforme n’héberge que ce site internet et le journal du lycée « Le Nautilus ».

La plateforme est en constant développement et ouverte à toutes suggestions. La plateforme est bien sûr ouverte à de nouveaux projets. Les projets ne sont pas nécessairement sous la responsabilité de l’équipe du site internet, mais devront veiller à respecter les règles légales, d'éthique et d’équité.

III. Une politique Open-Source

Le code source du site internet ljv.fr est entièrement disponible en ligne: vous pouvez le visualiser ou le télécharger depuis sa page Github. Le code est distribué sous la license MIT.

Si vous souhaitez suivre le développement de ce site internet ou y participer, vous pouvez vous abonner à sa page Github ou proposer vos propres modifications.
Si vous souhaitez vous investir d'une autre manière ou dans un autre projet, merci de prendre contact avec un membre de l’équipe du projet auquel vous vous intéressez.
